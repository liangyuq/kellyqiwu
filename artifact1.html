Artifact 1: Machine Learning Algorithms Framework

Introduction: This artifact is a visual framework that presents several foundational machine learning algorithms and explains how they are used in real-world applications.

Description: The infographic organizes six widely used machine learning algorithms including Linear Regression, Logistic Regression, Decision Trees, Support Vector Machines, K-Nearest Neighbors, and Naive Bayes. Each algorithm is presented with its learning type, common application domains, example use cases, and a short explanation of how it works.

Objective: The goal of this artifact is to help learners understand the differences between common machine learning algorithms in a simple and visual format. It provides a high-level overview that makes complex concepts easier to grasp for beginners.

Process: I first identified commonly used machine learning algorithms that represent key supervised learning approaches. I then summarized how each algorithm works using simple language and organized the information into structured sections. Finally, I designed the infographic layout in Canva to ensure the content is visually clear and easy to follow.

Tools/Technologies Used: Canva for infographic design, ChatGPT for drafting and simplifying explanations, and image export tools to prepare the PNG file for web display.

Value Proposition: This artifact demonstrates my ability to explain technical machine learning concepts in a clear and visually engaging format. It highlights skills in educational communication, content simplification, and visual design, which are important when presenting technical topics to broader audiences.

References: Concepts summarized from standard machine learning resources including introductory machine learning textbooks, educational articles, and academic course materials.
