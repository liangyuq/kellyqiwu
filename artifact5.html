Artifact 5: Balancing Performance, Efficiency & Explainability

Introduction: This artifact is an infographic that explores how different machine learning models balance performance, efficiency, and explainability.

Description: The infographic compares several models across domains such as NLP, computer vision, and tabular data. It highlights trade-offs between accuracy, speed, model size, and interpretability, and presents side-by-side comparisons, strengths and weaknesses, and recommendations for when to use each model.

Objective: The goal of this artifact is to help learners understand that no single model is best for all situations. It shows how model selection depends on the specific problem, constraints, and priorities.

Process: I reviewed different types of machine learning models and identified key metrics such as accuracy, inference speed, size, and explainability. I then organized the information into clear comparison sections and designed the infographic to make trade-offs easy to understand.

Tools/Technologies Used: Canva for infographic design, ChatGPT for drafting and simplifying explanations, and image export tools to prepare the PNG file for web display.

References: Content based on general machine learning knowledge and model evaluation concepts commonly used in NLP, computer vision, and tabular data analysis.
